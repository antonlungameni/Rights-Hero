# Namibian Legal Assistant

This project is a web-based legal assistant that helps answer questions about Namibian law.

## Features

- Ask a question about Namibian law and get an answer.
- Clear all previous questions and answers.
- Loading indicator when waiting for a response.

## Setup

1. Clone the repository.
2. Open the project in Visual Studio Code.
3. Run the project.

## Usage

1. Enter your question in the text area.
2. Click the 'Ask' button to submit your question.
3. Wait for the response. A loading indicator will be displayed while waiting.
4. The response will be displayed in the 'Messages' section.
5. Click the 'Clear' button to clear all messages.
